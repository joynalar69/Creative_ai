⌨️ টাইপিং স্পিড টেস্ট 📁 ফাইল ও ফোল্ডার অর্গানাইজেশন 🌐 সেফ ইন্টারনেট ব্রাউজিং 📄 ডিজিটাল সিভি (স্টুডেন্ট প্রোফাইল) 📝 অ্যাপ্লিকেশন রাইটিং 📊 মার্কশীট (এক্সেল) 🎨 পাওয়ারপয়েন্ট প্রেজেন্টেশন 🖼️ পোস্টার ডিজাইন 📢 ব্যানার ডিজাইন 📱 সোশ্যাল মিডিয়া পোস্ট ডিজাইন 🔖 লোগো ও ব্র্যান্ড আইডেন্টিটি 📁 পার্সোনাল ডিজাইন পোর্টফোলিও 🤖 এআই ইমেজ ক্রিয়েশন 🎨 এআই ফটো এডিটিং 🎙️ এআই ভয়েস স্টোরি 🎬 এআই শর্ট ভিডিও 🌐 পার্সোনাল ওয়েবসাইট 📄 সিম্পল HTML ওয়েব পেজ 💼 পোর্টফোলিও ওয়েবসাইট 📱 মিনি অ্যাপ UI ডিজাইন 📘 ফেসবুক পেজ সেটআপ 🎬 ইউটিউব চ্যানেল ব্র্যান্ডিং ✍️ এসইও কন্টেন্ট রাইটিং 📊 ডিজিটাল মার্কেটিং ক্যাম্পেইন 🧠 এআই স্টাডি হেল্পার 💼 ফ্রিল্যান্সিং প্রোফাইল 🎯 ফাইনাল ডিজিটাল স্কিলস পোর্টফোলিও

  • কম্পিউটার, ডিজাইন, এআই, ওয়েব, মার্কেটিং - ৫টি স্কিল
  • ফ্রিল্যান্সিং মার্কেটপ্লেসে একাউন্ট তৈরি
